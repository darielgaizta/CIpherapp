How to run:
	0. Pastikan Python sudah terinstal di komputer ya...
	1. Buka terminal, shell, atau CMD (Command Prompt) di direktori projek ini
	2. Aktifkan virtual environment
	   Windows	: >>> env\Scripts\activate.bat
	   MacOS	: >>> source env/Scripts/activate
	3. Jalankan program main.py
	   >>> python main.py

[NOTE] Untuk menghindari terjadinya error, jangan buka file Excel ketika sedang menjalankan program.><ds>88822</ds>
